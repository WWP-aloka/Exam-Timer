New Features: Per-Question Timer Mode: You can now toggle this mode in the Settings. When enabled, the timer represents the time for the current question only. Once the time for a question runs out, the app automatically plays a warning chime and advances to the next question, resetting the countdown. You can customize the Time per Question (e.g., 1 minute, 2 minutes, etc.) using a slider or direct input. Question Progress Grid: A new visual grid has been added below the main timer. It displays all question numbers (e.g., 1 to 50). Completed questions are dimmed, the current question is highlighted with a bold border and shadow, and upcoming questions are shown as outlines. This gives both invigilators and students a clear visual representation of the "complete set of questions" and current progress. Automatic Navigation: The timer "decreases one by one for each question" by resetting itself as you move through the set. The urgency indicator (red pulse) now triggers when there are only 10 seconds left for the current question, helping maintain a steady pace. Automatic Numbering System: The app now automatically generates a sequence of question numbers based on the Total Questions setting (defaulting to 50). The Question Grid dynamically renders these numbers, so you never have to manually label them. The Header and Main Timer automatically update to show "Question X of Y" as the exam progresses. Interactive Question Grid: The grid is now fully interactive. You can click on any question number to jump directly to it. When in "Per-Question Timer" mode, jumping to a question automatically resets the countdown for that specific question. Completed questions are visually dimmed, the current question is highlighted, and upcoming questions are outlined, providing an "at-a-glance" status of the entire exam. Enhanced Settings Control: Added a Number of Questions slider in the settings dialog, allowing you to quickly adjust the exam size (up to 200 questions). The app ensures the current question is always valid even if you reduce the total number of questions mid-exam. Visual Clarity: Added a "Question Grid" label and a progress counter (e.g., "12 / 50") above the grid for better context.
